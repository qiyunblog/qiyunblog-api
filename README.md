# qiyunblog-api
山河开发的一个基于SpringBoot+Mybatis前后端分离的个人博客

### 项目技术一览
本项目使用到的技术有:
应用层:Spring,SpringBoot,Swagger,Lombok,JWT
网关层:Nginx,Tomcat
用户层:HTML5,CSS3,Javascript,Jquery,Ajax/Axios,Vue.js
数据层:MySql,Mybatis
管理工具:Maven

本来想用基于Vue脚手架,然后使用ElementUI的,但是奈何实力不精,放弃了.
